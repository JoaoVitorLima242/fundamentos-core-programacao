# Exercícios

Exercícios de estudo organizados por **matéria** e por **tópico**. Cada exercício vive na
pasta do tópico correspondente. Os prefixos numéricos preservam a ordem didática.

## Estrutura

```
01-matematica-discreta-logica-formal/
├── 01-logica-booleana-proposicional/
├── 02-tabelas-verdade-operadores-equivalencias/
├── 03-conjuntos-relacoes-funcoes/
├── 04-combinatoria-contagem/
├── 05-inducao-matematica/
└── 06-teoria-dos-grafos/

02-design-clean-code/
├── 01-nomes-funcoes-legibilidade/
├── 02-coesao-acoplamento/
├── 03-separacao-responsabilidades/
├── 04-abstracao-encapsulamento/
├── 05-organizacao-em-modulos/
├── 06-solid/
└── 07-design-patterns-gof/
```

> Tópicos opcionais de matemática (probabilidade, estatística, álgebra linear, cálculo)
> serão adicionados só quando o domínio pedir.

## Como rodar um exercício

Cada exercício é autocontido. Ex.:

```bash
node 01-matematica-discreta-logica-formal/01-logica-booleana-proposicional/can-it-deploy-condicional-math/index.js
```
